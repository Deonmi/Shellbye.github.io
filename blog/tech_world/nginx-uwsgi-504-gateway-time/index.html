nginx uwsgi 504 gateway time-out

使用nginx做反向代理，然后结合uwsgi部署django项目，因为django中要访问远程数据库的一些数据，所以比较耗时，会导致出现504错误，参考了这个官方文档后，对nginx的设置进行了如下修改：

uwsgi_read_timeout 180;

然后就大功告成了。
